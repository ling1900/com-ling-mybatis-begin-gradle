mybatis 最基础示例，gradle 版。

===============================================

gradle 运行：

在 Tasks 中选择（ application 下）：

bootRun

虚拟机参数：

// https://github.com/spring-projects/spring-boot/issues/832#issuecomment-272780482

-Dspring.profiles.active=dev

注意：和 maven 版的参数写法不同。
